使用'hexo + github'搭建个人博客

Created2020-03-31updated2020-03-31

[TOC]

搭建环境

  • 安装Nodejs
  • 安装Git
  • 创建github账号
  • 本地创建文件夹，命名为Blog，博客相关文件存储在此文件夹下
  • 配置Git个人信息（配置完会提示重新登录）
  • 在Blog目录下修改_config.yml

  • 将hexo发布到github
  • 修改hexo配置_config.xml,
  • 修改其他内容去配置文件中对应位置修改。

发布博客

  • 创建一篇博客，会在Blog/source/_posts/下生成md文件。
